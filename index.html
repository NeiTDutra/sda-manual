MANUAL DE USO

Geral

O Sistema Despachante de Armas (SDA) é um sistema web para gerenciar clientes, usuários, departamentos e funcionários de departamentos, protocolos e documentos em PDF relacionados.

É um sistema para ser usado em escritórios despachantes de armas para auxiliar no processo de manipulação de grande volume de documentos rotativos.

Com os formulários do SDA - SYS é possível:

  • Criar, editar, anexar arquivos PDF, imprimir e excluir protocolos;
  • Criar, editar, anexar arquivos PDF e excluir clientes;
  • Criar, editar e excluir utilizadores;
  • Criar e excluir campos de documentos;
  • Criar e excluir departamentos;
  • Criar e excluir funcionários;

Este manual auxilia com detalhes, a entender como utilizar o Sistema Despachante de Armas.

Qualquer dúvida, sugestão ou crítica restante após a leitura do manual ou uso do SDA - SYS, utilize o formulário de contato direto com o time de desenvolvimento do sistema.


Login

Logo que abre o site do sistema a primeira página é a de login (fig.1), não sendo possível acessar os recursos sem ter utilizador e senha cadastrados. O cadastro de novos utilizadores é registrado apenas por administradores.

Até esta versão, existem dois níveis de utilizadores: ADMINISTRADORES E NÃO ADMINISTRADORES.


Efetuar o login é simples, digitar o nome de utilizador e a senha, e clicar em Entrar.(fig.2)


Ao efetuar login com sucesso, o sistema apresenta a página inicial dos recursos.(fig.3)

Criar Protocolo


Existem campos no formulário de protocolos que são do tipo selecionar, que são:

  • Campo Origem - Departamentos e Funcionários;
  • Campo Destino - Departamentos e Funcionários;
  • Campos Ad. Itens(Documentos) - Ítens;
  • Campo Cliente - Clientes;

Estes ítens devem ser inseridos primeiramente antes de gerar qualquer protocolo.


Para criar protocolo:

  1. Na parte superior clicar em "Novo Protocolo +".(fig.14)

  2. O formulário para novo protocolo carregará na tela.(fig.15)

  3. Após preecher, na parte inferior da página estão os botões de ação: Salvar e Cancelar.(fig.16)


Visualizar Protocolo

Para visualizar protocolo:

  1. Na tela que lista todos protocolos, clicar na linha do protocolo que desejar visualizar.(fig.17)

  2. Nesta tela também é possível pesquisar protocolos com filtro de pesquisa.(fig.18)

  3. Página de detalhes do protocolo.(fig.19)


    * Abaixo do protocolo detalhado estão os botões de ações para: imprimir, editar ou excluir protocolos.(fig.20)


Imprimir Protocolo

É possível imprimir somente o protocolo e também o protocolo incluindo todos arquivos PDF anexos.

Para imprimir somente o protocolo:

  1. Clique no botão "Imprimir Somente Protocolo".(fig.21)

  2. A caixa de diálogo de impressão do navegador se abrirá com o protocolo a ser impresso.(fig.22)


    Nesta ação pode-se salvar em PDF ou imprimir diretamente na impressora.


Para incluir todos anexos do protocolo na impressão basta clicar no botão "Imprimir Com Anexos".(fig.23)



Editar Protocolos

A edição/modificação de protocolos utiliza o mesmo formulário de criação.

Ao clicar no botão "Editar"(fig.24), o formulário abrirá com os campos preenchidos com os dados do formulário atual.(fig.25)




Salvar a edição segue o mesmo processo de salvar novo protocolo.


Excluir Protocolo

A exclusão de quaisquer itens no Sistema Despachante de Armas só pode ser efetuada por utilizadores administradores.

Para excluir protocolo, clicar no botão "Excluir" na página de detalhes de protocolo.(fig.26)


Para mais informações sobre exclusão de dados ver a seção "Funções restritas" deste manual.
